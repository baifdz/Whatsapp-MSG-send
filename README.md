# Whatsapp-MSG-send


Este script en JavaScript permite **enviar múltiples mensajes automáticamente** en una conversación abierta (por ejemplo, WhatsApp Web), escribiendo y enviando cada línea como un mensaje independiente.

> ⚠️ **Aviso**: Este código depende de la estructura del DOM de la página. Si la plataforma cambia su interfaz, el script puede dejar de funcionar.

---

## ✨ Características

- Divide un texto en múltiples líneas
- Envía cada línea como un mensaje individual
- Simula la escritura en un campo `contenteditable`
- Controla el tiempo entre mensajes
- Maneja errores si no hay una conversación abierta

---
## 🧪 Uso

1. Abre una conversación en la plataforma web compatible.
2. Abre la consola del navegador (`F12`).
3. Pega el código en la consola.
4. Modifica el texto dentro de `enviarScript(\`...\`)`.
5. Ejecuta y observa cómo se envían los mensajes automáticamente.
